Wordle Clone

Description

Create a Wordle game where users have to guess the correct word for the day.

Requirements

  1. Word will be 5 characters long (You can hardcode a list of words and choose a random word as Word of the day)
  2. User will get 6 guesses to guess the correct word
  3. On each guess highlight the letters which are:
    • In correct place
    • Present in the word but not in correct place
    • Not present in the word
  4. Display the keyboard on screen which can be used to enter the letters
  5. Display the word once user has guessed the correct word or failed to guess the word
  6. User can user keyboard/on-screen keyboard for inputting the letters

Source

You can checkout the game here.

Solution

State

  1. todayWord: Stores the today's word in useRef as this will not be updated
  2. position: An array of [x, y] indices used to traverse the wordle grid
  3. gridState: A 6x5 two dimensional array which shows all the letters and state of each letter to display to user. Each element can be an object representing the letter value and state of the letter. For eg. {letter: 'A', state: STATES.CORRECT}. We can either use useState or useRef to store this data, but using useRef has an advantage that we can directly mutate the state object and it will be re-rendered when position updates.
  4. hasGuessed: Value to represent the final state of the game. It can have 3 values:
    • null: This states that user has not guessed the word yet but have some guesses left
    • true: This states that user has guessed the word correctly
    • false: This states that user cannot guess anymore

Methods

onPressKey: This function is used to handle the user inputs
  1. Checks if user has pressed correct key
  2. If user has pressed Enter then the input word is checked against the Word of the day and status of letters are updated accordingly.
  3. If user has pressed Backspace it deletes the last letter from the user input
  4. If user has pressed an alphabet key the input is update with the pressed letter
This function is reused for handling inputs through Keyboard and inputs using on-screen keyboard.
